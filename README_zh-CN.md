# thinkjs monorepo

该仓库已迁移为基于 pnpm 的 monorepo 结构。

## 工作区结构

- `packages/thinkjs`: 原 ThinkJS 项目的源码、测试与包配置。

## 使用方式

在仓库根目录安装依赖：

```sh
pnpm install
```

执行 ThinkJS 包的 lint：

```sh
pnpm lint
```

执行 ThinkJS 包的测试：

```sh
pnpm test
```
