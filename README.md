# thinkjs monorepo

This repository has been migrated to a pnpm monorepo.

## Workspace Layout

- `packages/thinkjs`: original ThinkJS package source, tests, and metadata.

## Usage

Install dependencies from repository root:

```sh
pnpm install
```

Run lint for ThinkJS package:

```sh
pnpm lint
```

Run tests for ThinkJS package:

```sh
pnpm test
```
